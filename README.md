# Hough变换检测图像边缘

霍夫空间中每个点（斜率和截距）代表实际一条直线，霍夫空间中一条线代表实际世界中经过固定点的所有直线，
霍夫算法（投票制）：设定阈值，取霍夫空间中交于一点的线的数目大于阈值的所有点坐标，将所取的点坐标在实际世界中画出，即为边缘
